# Optimizator Godišnjeg Odmora (Vacation Optimizer)

## About / O aplikaciji

**English:**  
A web application designed to help you efficiently plan your vacation days by suggesting optimal schedules that maximize consecutive free days. The app takes into account weekends and Serbian public holidays.

**Srpski:**  
Web aplikacija dizajnirana da vam pomogne u efikasnom planiranju godišnjeg odmora, predlažući optimalne rasporede koji maksimizuju broj uzastopnih slobodnih dana. Aplikacija uzima u obzir vikende i državne praznike Republike Srbije.

## Features / Funkcije

**English:**

- Calendar visualization for the entire year
- Tracking of weekends and public holidays
- Manual selection of vacation days
- Automatic suggestions for optimal vacation plans:
  - Efficiency Plan: Best ratio of free days to vacation days used
  - Maximum Free Days Plan: Maximum consecutive free time
  - Shorter Vacations Plan: More, but shorter vacation periods
- Persistent storage of your selections
- Responsive design for both desktop and mobile

**Srpski:**

- Vizualizacija kalendara za celu godinu
- Praćenje vikenda i državnih praznika
- Ručni odabir dana godišnjeg odmora
- Automatski predlozi za optimalne planove odmora:
  - Plan Efikasnosti: Najbolji odnos slobodnih dana i iskorišćenih dana odmora
  - Plan Max Slobodnih Dana: Maksimum uzastopnog slobodnog vremena
  - Plan Kraćih Odmora: Više kraćih perioda odmora
- Trajno čuvanje vaših odabira
- Responzivan dizajn za desktop i mobilne uređaje

## How to Use / Kako koristiti

**English:**

1. Enter the number of vacation days you have available for the year
2. Click "Show Suggestions" to see recommended vacation plans
3. Click on any suggested plan to apply it to the calendar
4. Alternatively, manually select or deselect specific days by clicking on them
5. The system will track your remaining vacation days

**Srpski:**

1. Unesite broj dana godišnjeg odmora koji imate na raspolaganju za godinu
2. Kliknite na "Prikaži Predloge" da biste videli preporučene planove odmora
3. Kliknite na bilo koji predloženi plan da ga primenite na kalendar
4. Alternativno, ručno izaberite ili poništite izbor određenih dana klikom na njih
5. Sistem će pratiti preostale dane godišnjeg odmora

## Technologies Used / Korišćene tehnologije

- HTML5
- CSS3
- JavaScript (vanilla)
- Nager.Date API for public holidays (https://date.nager.at/)
- Local Storage for data persistence

## Installation / Instalacija

**English:**  
Simply open the `index.html` file in your web browser. No server or installation required.

**Srpski:**  
Jednostavno otvorite `index.html` datoteku u vašem web pretraživaču. Nije potreban server ili instalacija.
